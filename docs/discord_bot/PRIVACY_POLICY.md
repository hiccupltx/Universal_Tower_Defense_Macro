# Privacy Policy

Last Updated: 6/12/2026

This Discord bot is used to verify users' access to services through a verification process.

### Information We Collect

When you use the bot, we may collect and store:

- Discord User ID
- Discord Username
- Verification status
- Hardware Identifier (HWID) provided during verification
- Hashed versions of hardware identifiers
- Verification timestamps
- Basic bot usage logs for security and troubleshooting

### How We Use Information

The collected information is used to:

- Verify user eligibility and access.
- Prevent unauthorized account sharing.
- Detect fraud and abuse.
- Maintain the security and integrity of the verification system.
- Provide support when verification issues occur.

### Hardware Identifiers

Hardware identifiers submitted during verification are processed and stored in hashed form whenever possible. The hash is used solely for verification and anti-abuse purposes.

### Data Sharing

We do not sell, rent, or share personal information with third parties except:

- When required by law.
- To protect the security and integrity of the service.

### Data Retention

Verification data may be retained for as long as necessary to operate and secure the verification system.

### Data Deletion

Users may request deletion of their stored verification data by creating a ticket in our server and request to delete them.

### Security

Reasonable measures are taken to protect stored data from unauthorized access.
