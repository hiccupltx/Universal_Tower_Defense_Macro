## Setup

This macro use **Pixel Detection** so user must have exact the same graphic as other for the macro to work

### 💻 Window setup

- **Display resolution:** `1920x1080` recommended (`16x9` must)
- **Scale:** `100%`
- **HDR:** `Off`
- **Keyboard layout:** `qwerty`

### 🎮 Roblox setup

- **Graphic:** `1`
- **Fullscreen:** `Off`
- **Keyboard:** `Keyboard + Mouse`
- **FPS Cap:** `60`
- **Mouse sensitivity:** `1`
- **Chat Translation Language:** `English` (IMPORTANT)

### ♟️ Universal Tower Defense setup

#### Gameplay

- **Performance Mode:** `On`
- **Low Vfx Mode:** `On`
- **Auto Wave Skip:** `On`
- **Auto Skip Cutscenes:** `On`
- **Disable Placement Cutscene:** `On`
- **Disable Unrivaled Finisher:** `On`

#### Unit

- **Everything:** `Off` (IMPORTANT)

#### Enemies

- **Enable Enemies Animations:** `Off`
- **Show Enemy Info:** `Off`

#### Keybinds

- **Upgrade Unit:** `E` (IMPORTANT)
- **Sell unit:** `X` (IMPORTANT)
- **End Place:** `C` (IMPORTANT)
- **Macro (Play):** `Z` (IMPORTANT)


#### UI

- **UI Scale:** `100%` (IMPORTANT)

## Start Macro

There is already a built in **Coordinate** and **Strategy** but you can still config them yourself.

**Note:** The Default Strategy is only for 6 slots unlocked (**Bulma** at slot 5 and **Speedwagon** at slot 6)

### Config Team

- Config your team in `Config > Placements` with your matching team


### Config Tasks

- Setup your task `Task`
- **Route:** Main Task then do Custom task until timer for Main Task is reset
- Then just start your macro with **F2**
